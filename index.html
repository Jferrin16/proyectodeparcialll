Nivel 2: CSS

Descubre cómo aplicar colores, estilos y diseño visual con CSS.

Nivel 3: JavaScript

Haz tus páginas interactivas con JavaScript, el lenguaje que da vida a la web.

Nivel 4: Python

Domina la lógica y automatiza tareas con Python, uno de los lenguajes más populares y versátiles.
